Restful API library system using Java 17 and Spring Boot.
Docker is used to containerize Spring Boot application and MySQL database. 
Frontend UI uses React

After running docker, just direct to http://localhost:3000/ to access
Tasks performed in this system
1. Register borrower 
2. Register book 
3. Display all books 
4. Borrow book 
5. Return book
6. Update borrower details
7. Delete borrower
8. Delete book
